# Programas Java: Lectura de Cadena y Ejecución de Proceso

Este repositorio contiene dos programas en Java relacionados con la lectura de datos desde la entrada estándar y la ejecución de otro programa:

---

## 1. Lectura de cadena hasta asterisco: `LecturaAsterisco.java`

### Objetivo
- **FR1:** Leer una cadena de caracteres desde la entrada estándar hasta que se ingrese `*`.
- **FR2:** Mostrar la cadena completa leída.
- Implementa control de errores en la entrada del usuario.

### Compilación
```bash
javac LecturaAsterisco.java
