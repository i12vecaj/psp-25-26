# Programa Java: EjecutaAnalizador

Este programa ejecuta otro programa Java (`AnalizadorArgumentos`) que analiza argumentos pasados a `main()` y devuelve códigos de salida según ciertas reglas.

---

## Objetivo
- Ejecutar el programa `AnalizadorArgumentos` desde Java.
- Interpretar el código de salida y mostrar un mensaje claro en pantalla.
- Implementa control de errores en la ejecución del proceso.

---

## Compilación
```bash
javac EjecutaAnalizador.java
