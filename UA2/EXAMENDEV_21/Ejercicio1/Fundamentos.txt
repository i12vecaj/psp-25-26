y el metodo start() es el que llama al metodo run() para iniciar el proceso de los hilos

Ejemplo de run(), en este solo se puede crear un hilo haciendo que sea menos utilizable en proyectos reales

public class HiloSimple1 extends Thread {
	private int id;
	private int iter;
	
	public HiloSimple1 (int id, int iter)
	{
		this.id = id;
		this.iter = iter;
	}
	
	public void run()
	{
		for (int i = 0; i < iter; i++)
		{
			System.out.println("Iteración " + i + " dentro del hilo " + id);
			try 
			{
				Thread.sleep(1000);
			} 
			catch (InterruptedException e) 
			{
				e.printStackTrace();
			}
		}
		
		System.out.println("El hilo " + id + " ha terminado su ejecución");
	}
}
