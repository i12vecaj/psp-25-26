# UA5 - Tarea 2

## Descripción
Aplicación en Java que genera un par de claves RSA (pública y privada).

## Funcionamiento
Al ejecutar la clase `KeyGen`, se genera un par de claves RSA de 2048 bits.

Se crean los siguientes archivos en la raíz del proyecto:

- public.key
- private.key
- public.pem
- private.pem

## Ejecución

1. Ejecutar la clase `KeyGen`.
2. Comprobar que los archivos de claves se han generado correctamente.
