import java.util.Scanner;

    public static void main(String[] args) throws Exception {

        Scanner sc = new Scanner(System.in);

        System.out.println("Password:");

                "jdbc:mysql://localhost/app", "root", "root");

                + user + "' AND password='" + pass + "'";

            System.out.println("Acceso concedido");

            System.out.println("Acceso denegado");

Porque cualquier usuario que tenga acceso a la base de datos puede ver la contraseña, facilmente legible.

### Pregunta 2.3

¿Qué es el principio de mínimo privilegio? Pon un ejemplo en Java o en
bases de datos.
Es el principo que establece que un usuario solo debe tener los permisos justos y necesarios para ejercer su función en el programa.
Una forma común de establecer este principio es mediante el uso de roles, donde un posible usuario base solo tiene acceso a sus datos con sus credenciales, un posible gestor puede tener acceso a sus datos y los de los demás y por ejemplo, un super admin podría tener acceso a todo debido a su funcion de super usuario.
