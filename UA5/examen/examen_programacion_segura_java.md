import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.PreparedStatement;

import java.util.Scanner;

public class LoginSeguro {

    public static void main(String[] args) throws Exception {
        String dbUrl = System.getenv("APP_DB_URL");  //cogemos los datos de url, usuario y contraseña para guardarlos como 
        String dbUser = System.getenv("APP_DB_USER");// variables del entorno
        String dbPass = System.getenv("APP_DB_PASS");

             Connection con = DriverManager.getConnection(dbUrl, dbUser, dbPass);  //que coja las variables de entorno para

### Pregunta 3.3

Explica la diferencia entre:

-   KeyStore\
-   clave pública\
-   clave privada

1) KeyStore: es un almacen seguro donde se guardan claves y certificados protegidos por contraseñas, un lugar seguro donde guardar las claves en su totalidad.
2) Clave pública: se puede compartir, se usa para cifrar datos dirigidos de quien sea propietario la firma.
3) Clave privada: es secreta, se usa para ser como una llave para la clave pública y para firmar digitalmente.
