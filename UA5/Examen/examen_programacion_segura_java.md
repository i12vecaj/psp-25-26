### Pregunta 3.1

Explica qué es un certificado digital y para qué se utiliza.
Un certificado digital es un documento que ha sido validado por una organización de confianza (FNMT por ejemplo) que asigna una clave publica a una identidad. Se utiliza habitualmente como método de autenticación en algunas paginas y para realización de tramites entre otros.

### Pregunta 3.3

Explica la diferencia entre:

-   KeyStore\
-   clave pública\
-   clave privada

La clave pública es aquella que se comparte y que todo el mundo conoce mientras que la clave privada es conocida únicamente por el emisor y el receptor de un mensaje cifrado.

## BLOQUE 4 -- Criptografía RSA en Java

Analiza el siguiente código:

``` java
KeyPairGenerator keyGen = KeyPairGenerator.getInstance("RSA");
keyGen.initialize(1024);

KeyPair pair = keyGen.generateKeyPair();

PublicKey pub = pair.getPublic();
PrivateKey priv = pair.getPrivate();
```

### Pregunta 4.1

¿Qué tipo de criptografía utiliza RSA? Explica brevemente cómo funciona.
RSA usa criptografía asimétrica, es decir, a la hora de cifrar un mensaje, el emisor utiliza la clave publica para encriptar el mensaje mientras que el receptor usa la clave privada que previamente ha debido recibir para desencriptar dicho mensaje.
