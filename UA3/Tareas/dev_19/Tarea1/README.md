**Descripcion del ejercicio realizado**
- Primero se presenta una clase llamada 'PSP_UA3_EEjercicio_1_Visualizar_Informacion'
- Segundo, lo primero se presentan variables que posteriormente se utilizaran, tanto para leer datos, o para guardar los datos leidos
- Tercero, un control de excepciones donde dentro se encuentra toda la lógica de la clase
- Por un lado se comprueba si los datos que se han leído se refieren un una url o a una ip. Si es una url, se lee esa url y se muestra la siguiente informacion:
  - URL completa
  - Host
  - Puerto
  - Ruta
  - Ip asociada
- Por otro lado, si es una ip, se presenta la siguiente información:
  - Nombre del host
  - Dirección IP
  - Y si es alcanzable
- Y por último, una vez que se escriba la palabra "localhost", se cierra el bucle y se cierra el programa
